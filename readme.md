# firebase-movie-import

## Project setup
```
npm install --save firebase-admin
npm install --save dotenv

// or

npm install
```

## Compiles and run code
```
node index.js
```

## env Configuration
```
DATABASEURL=YOUR_DATABASE_URL_FIREBASE
```
